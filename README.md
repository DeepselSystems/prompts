# Prompts Repository

A collection of reusable prompts for LLMs, automation tasks.

## Purpose

A location to save, organize, and share prompts across projects and team members. Each prompt is documented with its purpose, usage instructions, and the actual prompt text for easy reference and reuse.

## Structure

Each prompt should be saved as a separate Markdown file (`.md`) in this repository. Follow the template below to ensure consistency and clarity.

## Prompt Template

Create a new `.md` file for each prompt with a descriptive name (e.g., `code-review-assistant.md`, `data-analysis-prompt.md`).

### Suggested Format

Each prompt file should include the following sections:

```markdown
# [Prompt Name]

## Purpose

A clear, concise description of what this prompt is designed to do and what problem it solves.

## Usage

Instructions on how to use this prompt, including:
- When to use it
- What inputs or variables need to be provided
- Any prerequisites or context required
- Expected output or results

## Variables

(If applicable) List any variables or placeholders that need to be replaced:
- `{variable_name}`: Description of what should replace this
- `{another_variable}`: Description

## Prompt

[Your actual prompt text goes here]
```
